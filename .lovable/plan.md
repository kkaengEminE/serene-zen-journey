

# 🧘 Zen 명상 앱 — 구현 계획

## 앱 흐름

### 1. 시작 화면
- 미니멀한 일본풍 디자인 (먹색 배경, 세리프 서체)
- **시작하기** 버튼 + **"Zen 스타일이란?"** 버튼
- Zen 설명 모달/페이지: 선(禪) 철학 간략 소개
- 은은한 파티클 또는 안개 효과

### 2. 심리 테스트 (5~6개 질문)
- 어두운 톤에서 시작, 질문마다 서서히 밝아짐
- 질문 예: 지금 마음 상태, 스트레스 수준, 원하는 감정 등
- 페이드 전환 애니메이션으로 질문 간 이동

### 3. 정원 산책 시퀀스 (핵심 경험)
- **React Three Fiber**로 3D 느낌 구현
- 자갈 정원(枯山水) → 스이킨쿠츠(水琴窟) 물방울 파티클 → 빛을 향해 전진
- 카메라가 앞으로 서서히 이동하는 느낌
- 심리 결과 기반 **힐링 문구 3~4개**가 떠오르며 지나감
- 안개/파티클이 점점 밝아지며 빛의 터널로 진입

### 4. 피날레 — 탁 트인 풍경
- 밝은 빛 플래시 후 광활한 자연 풍경 등장
- 랜덤으로: 지중해 바다 / 우유니 사막 / 푸른 언덕 중 하나
- CSS 그라데이션 + 3D 지평선으로 표현
- **"내면의 평화"** 메시지 + **다시 하기** 버튼

## 기술 구현
- **@react-three/fiber v8 + @react-three/drei v9**: 3D 정원, 파티클, 카메라 이동
- **Tailwind CSS**: UI 레이아웃, 색상 전환
- **Framer Motion**: 텍스트 페이드, 화면 전환
- 폰트: Noto Serif JP (제목), Noto Sans JP (본문)

## 디자인 톤
- 컬러: 먹색(#1a1a1a) → 회갈색 → 따뜻한 베이지 → 밝은 하늘색/흰색
- 질문 단계는 어둡고 고요, 정원은 따뜻하고 은은, 피날레는 밝고 탁 트인 느낌
- 미니멀 일본 미학, 여백 활용
